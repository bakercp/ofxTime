Detailed Description

Interval represents a timespan defined bounded by timestamps. The minimum Poco::Timestamp is guaranteed <= maximum Poco::Timestamp. Poco does not offer full support for Poco::Timestamp < 0. The behavior of Interval for values < 0 is undefined.

Comparison operators operate on one or both end points. Equality operators compare both end points. Comparison operators >, <, >=, <= compare only the minimum end point and are primarily useful for sorting.
