ofxTime

39  // A Period represents an abstract amount of time. In some cases a concrete
40  // number of microseconds can be calculated (e.g. for DateTimeFields such
41  // as MICROSECOND, MILLISECOND, SECOND, MINUTE, HOUR), but for some measures
42  // of time (e.g. HOUR, DAY, WEEK, MONTH, YEAR), it is only possible to
43  // calculate the equivalent number of microseconds relative to a known
44  // calendar start time. This is because concepts such as "MONTH" and "YEAR"
45  // are vary due to "leap" seconds, and other calendrical irregularities.
46  //
47  // Empty or NULL periods are those that have all fields set to zero.
